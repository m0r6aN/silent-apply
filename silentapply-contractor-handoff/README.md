# SilentApply Contractor Handoff Pack

## What This Is

SilentApply AI brand, motion, and visual governance materials.

## What You May Do

- Use canonical assets only
- Assemble pages within doctrine constraints
- Ask before introducing new visuals

## What You May NOT Do

- Create new icons or marks
- Add spinners, loaders, progress bars
- Use people, resumes, or job metaphors
- Improve animations
- Add excitement

## One Rule to Remember

If it feels persuasive, urgent, or clever -- it is wrong.

## Contractor Acceptance Clause

All work must comply with the SilentApply Doctrine.
Non-compliant assets will be rejected regardless of aesthetic quality.

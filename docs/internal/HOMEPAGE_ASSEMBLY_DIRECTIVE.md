# Homepage Assembly Directive

Owner: Codex / Frontend Agent
Goal: Assemble homepage using only canonical assets and doctrine rules.

## Layout Order (Top to Bottom)

1. Hero Section
2. SilentApply Value Explanation
3. How It Works (Calm Steps)
4. Candidate Control / Trust
5. Footer (Minimal)

## Hero Assembly Rules

### Assets

- Use canonical homepage hero
- Overlay copy only -- no embedded text in image
- Dot must be visually dominant

### Spacing

- Vertical padding: generous
- Copy never overlaps hero focal point
- Mobile crop must preserve center presence

### Copy Placement

- Headline: left or center-left
- Subheadline: single sentence max
- CTA: secondary weight (not dominant)

- No buttons that scream
- No animated entrances
- No auto-scroll

## Interaction Rules

- Hover states: opacity or underline only
- CTA emphasis: typography, not motion
- Scroll: native browser behavior only

If the page feels like it is trying, it is wrong.

## Enforcement Checklist

Before merge:

- [ ] No non-canonical visuals
- [ ] No forbidden motion patterns
- [ ] Dot present and dominant
- [ ] Hero reads as already working
- [ ] Mobile layout preserves calm

# Canon Compliance Checklist

## Single Source of Truth

CANON.md is the single source of truth for tone, public copy, and recruiter-facing behavior.
If any document conflicts, CANON.md controls. Do not merge conflicting copy.

## Copy Review Checklist

- Copy is calm, declarative, bounded.
- No urgency, persuasion, or excitement framing.
- No forbidden language (see AGENTS.md), no "powered by AI", no model names.
- No claims beyond shared profile fields or resume content.
- Public surfaces show no monetization cues.

## Workflow Guardrail

- Review new or edited copy against CANON.md and this checklist.
- If uncertain, remove the copy or narrow it until it is strictly factual.
